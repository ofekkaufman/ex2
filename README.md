# Project Name

**תיאור קצר:**  
פרויקט זה כולל מערכת לניהול מכשירים (Devices) ומשתמשים, עם פונקציות שונות הממומשות בקבצי CPP ו-H.

## רשימת קבצים

- `Devices.h` – הגדרות ומחלקות הקשורות למכשירים.  
- `Devices.cpp` – מימוש הפונקציות של Devices.  
- `User.h` – מחלקת User והגדרות הקשורות למשתמש.  
- `User.cpp` – מימוש הפונקציות של User.  
- `main.cpp` – נקודת הכניסה של התוכנית.  
- קבצי בדיקה (אם קיימים, לדוגמה `testDevices.cpp`) – בדיקות לפונקציות של Devices ו-User.

## הוראות קומפילציה והפעלה

1. פתח את הפרויקט ב-Visual Studio (או בכל IDE אחר שתומך ב-C++).  
2. הוסף את כל קבצי ה-CPP וה-H לפרויקט.  
3. קומפל והרץ את `main.cpp`.  
4. תוכל להריץ גם את קבצי הבדיקה בנפרד כדי לבדוק פונקציות ספציפיות.

## הערות נוספות

- כל שינוי בקבצי `Devices` או `User` ישפיע על התוכנית הראשית.  
- אין צורך לשנות את קבצי הבדיקה אם רוצים לשמור על תקינות הבדיקות.
